## 2. 트랙 C — 이도원 (R1) 🎨 프레젠테이션 승격 ⭐크리티컬 패스

게임의 #1 격차. **이 트랙의 시간을 최우선 보호**한다. A·B는 C를 돕는 방향으로 붙는다.

### 2-1. 아트 디렉션 — 도형 프리미엄 룩 (P0)

현재 렌더는 원·호·라인 프리미티브(`render/palette.ts`·`render/spellRenderer.ts`·`render/roomBackdropConfig.ts`).
스프라이트 없이도 **일관된 연출 규칙**으로 프리미엄 룩을 만든다 (레퍼런스: Geometry Wars · TxK · Rez):

- 통합 블룸/글로우 파이프라인(ADD 블렌드 일관 적용), 원소별 팔레트 대비·채도 강화
- 잔상 트레일(플레이어·투사체·주문), 배경 그리드 깊이감(패럴랙스·명멸)
- 주문 임팩트 프레임: 발동 섬광·충격파·화면 비네트
- 수치·규칙은 config 상수로 분리 (총괄이 밸런스와 함께 조정 가능하게)

### 2-2. 게임필 / juice (P0)

`feat/enemy-juice`(미병합, commit `aeb2593` "피격·공격·죽음 트윈") 완성·병합 후 확장:

- 히트스톱(타격 시 수 프레임 정지), 화면 셰이크 강약 계층, 넉백·스쿼시
- 적 사망 연출(파편·플래시), 피격 색 점멸, 데미지 팝업 폴리시
- 영창 진입 슬로모션 연출 강화(비네트·시간 왜곡 라인)

### 2-3. 사운드·타이틀 마감 (P1)

- 원소별 시전음·타격음·보스 BGM 밸런스 (기존 `audio/gameAudio.ts` 위에)

## 3. 트랙 A — jaepaly (총괄/R3) 🛡 기능 완성도

"이미 있는 게 흔들리지 않는다." C가 얹을 **안정 기반**을 만든다.

- [x] **오토 ≤40% 라이브 실측** (2026-07-20): 각인 2(위력80)+정령 2 빌드 오토 DPS 7.33 vs 수동 DPS 26.67 = **27.5% (게이트 통과)**. 기본공격 격리·불사 더미로 측정, 회귀 산술과 방향 일치·여유
- [x] **96조합 전수 QA** (2026-07-20): 8원소×12폼 96 + effect/이중원소 35 + 크기×속도 96 = **227회 시전 무크래시·콘솔 무오류**
