# 폭염 저주 설계 제안

> 상태: 구현 전 검토 초안  
> 목적: 총괄 승인 후 별도 브랜치와 후속 PR에서 구현  
> 범위: 폭염 저주 단일 항목의 의도, 선정 조건, 지속 피해·냉각 규칙 및 검증 기준  
> 공통 선택 규칙: [CURSE_SELECTION_PROPOSAL.md](./CURSE_SELECTION_PROPOSAL.md)

## 1. 제안 목적

`폭염`은 물·얼음·바람 원소 영창을 사용해 지속적인 환경 피해를 관리하게 만드는 원소 카테고리 저주다.

방의 극심한 열기는 플레이어의 체력을 지속적으로 감소시킨다. 물·얼음·바람 원소 영창을 정상적으로 발동하면 체력을 회복하고 일정 시간 열기 피해를 받지 않는다.

폭염은 비친화 원소의 위력을 약화하거나 사용을 차단하지 않는다. 플레이어는 원하는 영창을 계속 사용하면서 열기 피해를 감수하거나, 친화 원소를 섞어 체력과 냉각 시간을 관리할 수 있다.

## 2. 암전과의 원소 친화 문법

폭염은 기존 원소 편향 약화와 다른 `특정 원소 친화` 기믹을 사용한다.

```text
암전
- 친화 원소: 빛·불꽃
- 친화 보상: 시야 확장

폭염
- 친화 원소: 물·얼음·바람
- 친화 보상: 체력 회복 및 열기 피해 면역
```

두 저주 모두 특정 원소를 사용하지 않았다는 이유로 해당 원소의 공격력을 직접 조정하지 않는다. 친화 원소를 사용하면 환경 문제에 능동적으로 대응할 수 있는 새 선택지를 제공한다.

## 3. 저격할 플레이 습관

폭염은 직전 런에서 물·얼음·바람 수동 영창을 거의 사용하지 않은 플레이어에게 높은 행동 점수를 부여한다.

```text
냉각 원소 사용 비율
= 물·얼음·바람 친화 수동 영창 수 / 전체 정상 수동 영창 수

폭염 원시 행동 점수
= 1 - 냉각 원소 사용 비율
```

수동 영창 표본이 부족한 런에서는 실제 비율을 그대로 쓰지 않고 공통 중립값과 혼합하거나 중립값으로 처리한다. 최소 표본과 정규화 구간은 플레이테스트 데이터로 확정한다.

시퀀스는 내부에 물·얼음·바람 원소 behavior가 하나라도 실제 실행됐다면 친화 수동 영창 1회로 기록한다. 친화 behavior가 여러 개 있어도 한 plan은 분자와 분모에 각각 최대 1회만 반영한다.

## 4. 기본 피해와 냉각 수치

초기 제안값은 다음과 같다.

| 항목 | 초기값 |
|---|---:|
| 폭염 지속 피해 | 최대 체력 `1%/초` |
| 방 진입 피해 유예 | `3초` |
| 친화 영창 즉시 회복 | 최대 체력 `10%` |
| 친화 영창 열기 면역 | `5초` |

냉각 영창 사용 후 5초 동안 피해를 받지 않고, 이후 10초 동안 노출되면 최대 체력 10%를 잃는다. 약 15초마다 친화 영창을 사용하면 폭염으로 인한 체력 손실을 대략 상쇄할 수 있다.

- 15초보다 자주 사용: 폭염 피해를 넘어 적 공격 피해까지 회복 가능
- 15초보다 늦게 사용: 체력이 점차 감소
- 전혀 사용하지 않음: 다른 피해가 없더라도 약 100초 후 사망 가능

정확한 수치는 구현 후 플레이테스트로 조정한다. 초기 검증 범위는 지속 피해 `0.75~1.25%/초`, 면역 `4~6초`로 둔다.

## 5. 폭염 피해 규칙

폭염 피해는 일반 적 공격이 아닌 환경 저주 피해로 처리한다.

- 플레이어 최대 체력 기준
- 쉴드 무시
- HP에 직접 적용
- 폭염 피해로 사망 가능
- 넉백 없음
- 적 피격 히트스톱 없음
- 일반 적 공격 피격 통계와 회피율 통계에서 제외
- 방 정리·보상 선택·방 전환·런 종료 중 정지

실제 HP 계산은 프레임 단위 연속 감소 또는 누적값으로 처리할 수 있다. 피해 숫자와 UI 피드백은 과도한 텍스트 생성을 막기 위해 1초 단위로 묶어 표시한다.

## 6. 냉각 회복과 면역

물·얼음·바람 친화 수동 영창이 정상 발동하면 다음 효과를 동시에 적용한다.

```text
현재 HP를 최대 체력의 10%만큼 회복
열기 피해 면역 시간을 5초로 설정
```

- 최대 체력을 초과해 회복하지 않음
- 폭염 피해뿐 아니라 적 공격 등 다른 원인으로 잃은 체력도 회복
- 면역 중 다시 친화 영창을 사용해도 체력 10% 재회복
- 면역 시간은 누적하지 않고 5초로 갱신
- 체력이 가득 차 회복량이 0이어도 면역은 적용
- 방 종료 후 회복된 체력은 유지

친화 영창을 반복해 체력을 완전히 회복하는 플레이는 허용한다. 이는 저주를 우회하는 악용이 아니라 물·얼음·바람 원소를 적극적으로 사용한 보상이다. 플레이어는 회복을 위해 마나와 영창 기회를 사용하며, 전투를 지연하는 동안 적 공격 위험도 감수한다.

## 7. 전투 시간 기준

폭염 피해, 진입 유예와 냉각 면역은 실제 벽시계 시간이 아니라 전투 시간 기준으로 계산한다.

- 영창 입력 및 해석 슬로모션 중에는 피해와 면역 시간도 같은 비율로 느려짐
- 보상·방 전환·런 정지 중에는 피해와 타이머 정지
- 정상 전투 중에는 일반 속도로 진행
- 시퀀스 무적 중에는 기존 시퀀스 무적 규칙에 따라 폭염 피해 차단

실제 시간으로 계산하면 긴 문장을 입력하는 사용자에게 별도의 타자 속도 패널티가 생기므로 사용하지 않는다.

## 8. 단일 영창과 시퀀스 판정

### 8.1 단일 영창

- 물·얼음·바람 정상 수동 영창: 냉각 1회
- 감쇠 시전: 실제 발동하므로 냉각 1회
- 정상 cache/fallback: 냉각 1회
- 불발·침묵 차단·마나 소비 실패: 미적용
- 정령·각인·기본 공격·자동 공격: 미적용

### 8.2 시퀀스 영창

대표 원소 하나만 보지 않고 시퀀스 내부의 실제 실행 behavior 전체를 검사한다.

- form behavior의 주 원소와 보조 원소 판정
- 원소가 지정된 move behavior 판정
- wait 및 비원소 behavior 제외
- 첫 물·얼음·바람 behavior가 실제 실행되는 순간 냉각 발동
- 한 plan에 친화 behavior가 여러 개 있어도 회복·면역은 1회만 발동
- 친화 behavior 실행 전에 시퀀스가 중단되면 발동하지 않음

이 규칙은 암전의 빛·불꽃 시퀀스 판정과 동일한 원소 친화 공통 문법으로 사용한다.

## 9. 시각 및 UI 구현 방향

세부 이펙트는 구현 단계에서 제작하고 플레이테스트한다. 설계 단계에서는 전달해야 할 정보만 고정한다.

- 방 진입 시 `저주: 폭염`과 물·얼음·바람 대응법 안내
- 방 전체가 매우 덥고 위험하다는 인상을 주는 색조와 열기 왜곡
- 지속 피해가 적 공격과 다른 환경 피해임을 구분
- 냉각 발동 시 회복과 면역 시작을 명확히 표시
- HUD에 남은 냉각 면역 시간 표시
- 면역 종료 1초 전부터 시각·음향 경고
- 이펙트가 적 공격 예고와 위험 표시를 가리지 않음

안내 문구 초안:

```text
저주: 폭염
타오르는 열기가 생명력을 앗아갑니다
물·얼음·바람 영창으로 몸을 식히세요
```

## 10. 기존 시스템과의 상호작용

- 원소 피해량과 기존 원소 내성은 폭염 때문에 추가 변경하지 않음
- 친화 영창은 적중 여부와 무관하게 정상 발동 시 냉각 적용
- 감쇠 시전도 정상 발동했다면 냉각 적용
- 시퀀스는 실제 친화 behavior 실행 시 plan당 1회 적용
- 시퀀스 무적은 폭염 피해도 차단
- 폭염 회복은 일반 체력 회복으로 처리하되 폭염방 자체에는 치유 감소가 없음
- 폭염 피해와 회복은 일반 적 피격·회피 통계를 오염시키지 않음

## 11. 플레이테스트 항목

1. 물·얼음·바람을 사용하지 않을 때 폭염이 실제 생존 압박으로 느껴지는가
2. 10% 회복과 5초 면역이 친화 원소 사용의 명확한 보상으로 느껴지는가
3. 약 15초마다 한 번 사용하는 손익 기준이 실제 전투 리듬에 적절한가
4. 친화 영창 반복을 통한 완전 회복이 의도된 대응법으로 받아들여지는가
5. 쉴드 무시와 환경 피해가 UI에서 충분히 구분되는가
6. 시퀀스의 주·보조·move 원소가 정확히 판정되고 plan당 한 번만 발동하는가
7. 영창 슬로모션 중 타자 속도 때문에 부당한 추가 피해를 받지 않는가
8. 면역 종료 경고가 전투 중 식별 가능한가
9. 폭염 피해 숫자와 열기 효과가 다른 위험 예고를 가리지 않는가
10. 물·얼음·바람 사용 비율 기반 가중치가 암전과 공정하게 경쟁하는가

## 12. 승인 요청 사항

1. 폭염을 물·얼음·바람 원소 친화형 `element` 카테고리 저주로 추가
2. 최대 체력 `1%/초`의 쉴드 무시 환경 피해와 사망 허용
3. 친화 영창 시 최대 체력 `10%` 회복 및 `5초` 열기 면역
4. 방 진입 후 `3초` 피해 유예
5. 친화 영창 반복을 통한 완전 회복 허용
6. 피해·유예·면역을 전투 시간 기준으로 계산
7. 시퀀스 내부 친화 behavior의 실제 실행 시 plan당 한 번 발동
8. 물·얼음·바람 수동 영창 사용 비율을 후보 행동 지표로 사용
9. 시각 효과의 세부 구현과 수치 미세조정은 승인 후 플레이테스트에서 진행

승인 전에는 구현에 착수하지 않는다. 승인 과정에서 변경된 항목을 이 문서에 반영한 뒤 별도 브랜치에서 구현한다.
