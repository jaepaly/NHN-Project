## 8. wall·orbit 폼 확정 기획 (2026-07-20 총괄 — Issue #34 답변)

> 원칙: **기존 시스템과 역할이 겹치지 않는 정체성**을 하나씩만 준다.
> wall = 전장 설치형 공간 통제(투사체 차단이 핵심 가치), orbit = 플레이어 밀착 지속 견제(접촉 피해).
> 아래 수치는 전부 임시값 — 구현 시 config 상수로 두고 7/25 통합 밸런스에서 조정.

### 8-1. wall — 설치형 원호 방벽

- **정체성**: `shield` 효과(플레이어 부착 흡수막)와 다른 **전장 설치물**. "벽을 세워 막는다" 계열 문장이 damage/control과 결합될 때의 공간 통제 폼.
- **발현 조건**: `effect`가 `damage` 또는 `control`일 때만 폼이 실체화된다.
  `shield+wall` 판정(예: "나를 지켜줘")은 **현행 유지** — applySpellEffect의 shield 분기(보호막 수치)가 우선하고 벽은 설치되지 않는다 (기존 보호막 시스템과 중복 금지, 여유 시 벽 모양 연출만 얹는 것 허용).
- **생성 위치·방향** (target별):
  - `self`: 플레이어에서 최근접 적 방향 90px 지점, 진행 방향에 수직인 원호
  - `enemy`/`area`: 최다 적 방향 — 기존 `densestDirectionalTarget` 재사용, 그 방향 전방 90px
- **크기**: size별 호 길이 small 120 / medium 160 / large 220 / huge 300px, 두께 14px
- **지속**: speed별 slow 4s / normal 3s / fast 2.2s (느릴수록 오래 — zone과 동일 규칙)
- **판정**:
  - **적 투사체**: 접촉 시 소멸 (핵심 가치 — 슈터·보스 볼리 대응)
  - **적 이동**: 비보스는 통과 불가(충돌 클램프). **보스는 통과 가능**하되 통과 순간 1회 둔화 40%/1.5s (보스 봉쇄 불가 원칙 — cage의 감금과 차별)
  - **effect 조합**: `damage+wall` = 적 접촉 시 power×0.5 피해(적당 1회) / `control+wall` = 접촉 시 기존 control 둔화 부여(피해 없음)
- **렌더**: 원소 팔레트 원호 스트로크 + 가장자리 파티클 (외부 이미지 없음)

### 8-2. orbit — 플레이어 궤도 회전체

- **정체성**: 근접 지속 견제. `summon`(따라다니며 발사) · `nova`(1회 폭발) · `zone`(고정 장판)과 구분 — **발사 없이 접촉 피해만**.
- **발현 조건**: `effect`가 `damage` 또는 `control`일 때. 그 외 effect는 기존 효과 처리 우선(폼 미발현).
- **중심**: **항상 플레이어** (target 무관 — v1 단순화. enemy/area 판정이어도 플레이어 중심)
- **회전체 수**: size별 small 2 / medium 3 / large 4 / huge 5
- **반경·속도**: 반경 90px 고정, 각속도 speed별 slow 0.9 / normal 1.2 / fast 1.6 회전/s
- **지속**: 4s 고정 (speed는 각속도에만 반영)
- **피해**: 회전체 접촉(반경 16px) 시 **power×0.35**, **동일 적 재타격 쿨다운 0.8s** (지속 4s 동안 적당 최대 5회 — 밀착 유지 시 이론 상한 ×1.75, 실전 이탈 감안한 수치)
  - `control+orbit` = 접촉 시 둔화만 부여(피해 없음)
- **적 투사체 차단 없음** (wall과 역할 분리)
- **렌더**: `SpiritOrbView` 스타일 발광 구체 재사용 가능 — 정령 궤도(반경 68px)와 겹쳐 보이지 않게 반경·크기 차등
- **오토 게이트 무관**: 정령 고유 폼 표(§ PROGRESSION_DESIGN)에 orbit·wall은 없다 — 수동 전용 폼. 다타격 배율은 재타격 쿨다운으로 상한이 잡힌다. 오토 시전 피해는 비반올림 정확값을 쓰므로(§1 게이트 항목) 향후 오토에 편입해도 산술 게이트가 유지된다.

### 8-3. 공통
